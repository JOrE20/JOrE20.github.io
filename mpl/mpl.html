Money Programming Language (MPL)

Money Programming Language (MPL) is an esoteric programming language originally created as a personal project on 3 October 2024 by JOrE20. The reference JavaScript implementation is provided by MPLJS.js. It's more an esoteric language rather than a golfing language. The stack is just designed to save registers in like assembly, not to make a stack-based language. The original 2024 design isn't ever had syscalls (they was just single-digits IO-releated operators), I originally designed syscalls for simple IO. Open the MPL Playground. Despite its simplicity, MPL programs can often be surprisingly short. The last version is v0.1.

Language overview

Programs are parsed one character at a time. Any unrecognized character is ignored. The language contains four general-purpose registers (R1–R4), a stack, and a collection of system calls.

Registers R1, R2 and R3 are initialized to 0. Register R4 is initialized to 1. Characters with code points greater than 161 are interpreted as their code point minus 151.

Registers

Register Initial value Description
R1 0 General-purpose register.
R2 0 General-purpose register. Most system calls use R2 as their primary operand.
R3 0 General-purpose register.
R4 1 General-purpose register. Used as the default increment/decrement value.

Instruction set

Instruction Description
$Increase R1 by R4.
-Decrease R1 by R4.
[...]Execute the enclosed code exactly R2 times.
{...}Execute the enclosed code while R2 is non-zero.
q...*Loop with R2 taking every value from R1 through R3 (inclusive).
L...*Loop with R2 taking every value from 1 through R1 (inclusive).
t...*Loop with R2 taking every value from 0 through R1-1 (inclusive).
@Set R1 to 0.
G...*Execute the enclosed code only if R2 is non-zero.
&...*Execute the enclosed code only if R2 is zero.
Z...*Execute the enclosed code only if R1 equals R2.
T...*Execute the enclosed code only if R1 does not equal R2.
!Copy R1 into R2.
/Copy R2 into R1.
;Copy R3 into R1.
'Copy R1 into R3.
~Copy R3 into R2.
`Copy R2 into R3.
=Copy R1 into R4.
QSet R4 to 1.
vLoad language major version to R1 and minor version to R2.
WCopy R2 into R4.
EAdd R2 to R4.
SSubtract R2 from R4.
NPrint the next command-line argument. R2 becomes 0 if nothing printed (no arguments left), otherwise 1.
IOutput a newline.
.Shortcut for system call 0 (output the ASCII character whose code is stored in R2).
VPrint the values of all four registers for debugging.
P f u cRead the next source character and store its Unicode code point into R1, R2, R3 or R4 respectively.
K F U CRead a decimal integer from the source and store it into R1, R2, R3 or R4 respectively.
numberInvoke the system call with the specified numeric ID.
"..."Output the enclosed string literally.
,No operation. Useful for terminating numeric literals.
?Push all registers to stack as R1 R2 R3 R4.
ePop all registers from stack as R1 R2 R3 R4.

System calls

ID Name Description
0Output ASCIIOutput the ASCII character whose code is stored in R2. Equivalent to ..
1argcOutput the number of command-line arguments.
2Input lengthOutput the total length of the null-separated input.
3Read characterRead the next input character to R2.
4RandomSet R2 to a random integer in the range [0, R2).
5Output decimalOutput R2 as a decimal integer.
6Output baseOutput R2 using base R3.
7Read decimalRead a decimal integer from the input into R2.
8Read baseRead an integer in base R3 from the input into R2.
9Greatest common divisorReplace R2 with gcd(R2,R3).
10MultiplyReplace R2 with R2 × R3.
11DivideReplace R2 with floor(R2 / R3).
12ModuloReplace R2 with R2 mod R3.
13Push R1Push R1 onto the stack.
14Push R2Push R2 onto the stack.
15Push R3Push R3 onto the stack.
16Push R4Push R4 onto the stack.
17Pop R1Pop the top value from the stack into R1.
18Pop R2Pop the top value from the stack into R2.
19Pop R3Pop the top value from the stack into R3.
20Pop R4Pop the top value from the stack into R4.
21Peek R1Copy the top value of the stack into R1 without removing it.
22Peek R2Copy the top value of the stack into R2 without removing it.
23Peek R3Copy the top value of the stack into R3 without removing it.
24Peek R4Copy the top value of the stack into R4 without removing it.
25DiscardRemove the top value from the stack.
26DuplicateDuplicate the top value on the stack.
27SwapSwap the top two values on the stack.
28MoveMove R2th element from top to R3th (0 = top) from stack.
